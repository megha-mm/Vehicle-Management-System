# Vehicle Management System

A console-based Vehicle Management System developed in C++ to manage vehicle inventory efficiently. The project implements Object-Oriented Programming concepts and file handling to perform operations such as adding, viewing, searching, updating, deleting, and purchasing vehicles.

### Key Features

* Role-based access (Admin/User)
* Vehicle inventory management
* Search functionality
* Update and delete operations
* Vehicle purchase management
* Persistent data storage using text files

### Technologies Used

* C++
* Object-Oriented Programming (OOP)
* File Handling (ifstream & ofstream)

This project demonstrates the practical application of C++ programming, data management, and basic software design principles.
